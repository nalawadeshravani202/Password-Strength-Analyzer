# Password Strength Analyzer

## Description

A Flask-based Password Strength Analyzer that evaluates password strength based on:

- Password Length
- Uppercase Letters
- Lowercase Letters
- Numbers
- Special Characters
- Common Password Detection
- Password Reuse Detection
- Secure Password Generator

## Technologies Used

- Python
- Flask
- SQLite
- bcrypt
- HTML
- CSS
- JavaScript

## Installation

```bash
pip install -r requirements.txt
python app.py
